The Cocktail DB:

The Cocktail DB:

The Cocktail Database is solely where our data comes from. The file DBScraper.py in the Mixopedia repository hits this API. It first gets a list of all of the cocktails available from The Cocktail Database, then requests the information for all available cocktails, and stores the results in a pickle file, cocktails.pkl. The scraper then retrieves images from the web for each cocktail and ingredient, and saves them to the corresponding /static/images directory.
